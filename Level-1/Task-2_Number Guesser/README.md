# Number Guessing Game

A simple command-line game built using Python where the user tries to guess a randomly generated number.

## Features
- Random number generation between 1 and 100
- Limited number of attempts
- Feedback after each guess (Too high / Too low)
- Input validation for non-numeric values

## How to Run
1. Open the folder containing `guessing_game.py`
2. Run the following command in terminal or CMD:

```bash
python guessing_game.py
